# Proyecto de Grado 2: Web frontend dermoapp

## Desplegar a heroku

### Login a Heroku

```bash
heroku login
```

### Agregar remote

```bash
heroku git:remote -a grupo4-tesis-front
```

### Desplegar

```bash
git push heroku master
```
